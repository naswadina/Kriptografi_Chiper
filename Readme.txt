# Judul Proyek: PROGRAM DEKSTOP KRIPTOGRAFI

## Deskripsi
Program ini menggunakan beberapa metode kriptografi seperti Vigenere, Playfair, dan Hill untuk mengenkripsi dan mendekripsi teks.

## Fitur
- Enkripsi dan dekripsi teks menggunakan berbagai cipher.
- Antarmuka pengguna berbasis GUI dengan Tkinter.

## Instalasi
1. Pastikan Jupyter Notebook sudah terinstal di komputer Anda.
2. Salin kode program ke dalam sel baru di Jupyter Notebook.
3. Jalankan sel tersebut untuk memulai aplikasi.

## Menjalankan Program
1. Buka Jupyter Notebook dan buat notebook baru.
2. Salin dan tempel kode program ke dalam sel di notebook.
3. Jalankan sel untuk menjalankan aplikasi.

## Antarmuka GUI
- **Input Teks**: Tempatkan teks yang ingin Anda enkripsi atau dekripsi.
- **Unggah File**: Klik tombol "Unggah File" untuk memilih file yang diinginkan.
- **Kunci**: Masukkan kunci enkripsi (minimal 12 karakter).
- **Pilih Cipher**: Pilih metode enkripsi (Vigenere, Playfair, atau Hill).
- **Enkripsi/Dekripsi**: Klik tombol "Enkripsi" atau "Dekripsi".
- **Hasil**: Hasil akan ditampilkan di area output di bawahnya.

## Validasi Input
- Kunci harus terdiri dari setidaknya 12 karakter.
- Kunci hanya boleh mengandung huruf alfabet (A-Z).
